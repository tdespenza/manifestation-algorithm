# Governance

This project is currently maintained by Tyshawn Despenza ("Maintainer").

## Roles and Responsibilities

### Maintainer

The Project Maintainer has final decision-making authority over the project. They are responsible for:

- Setting the technical direction.
- Reviewing and merging pull requests.
- Managing releases.
- Enforcing the Code of Conduct.

### Contributors

Contributors are community members who submit issues, pull requests, or documentation updates.

## Decision Making

Decisions are made through consensus when possible, but the Maintainer has the final say in case of disagreements.

## Changes to Governance

This governance model may be updated by the Maintainer as the project grows.
